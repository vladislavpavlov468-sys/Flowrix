Flowrix3D CRM — система заказов и управления 3D-печатью

Flowrix3D CRM — это веб-приложение на Flask, предназначенное для оформления заказов на 3D-печать и управления ими через встроенную CRM-систему.

Проект объединяет систему онлайн-заказов, CRM для управления заказами, личный кабинет пользователя, REST API и хранение файлов 3D-моделей.

Цель проекта — разработать полноценное веб-приложение, которое позволяет пользователям оформлять заказы на 3D-печать, загружать STL-файлы моделей, отслеживать статус выполнения заказа, а администратору управлять заказами через CRM-панель и взаимодействовать с системой через API.

Клиент:

регистрация и авторизация
создание заказа
загрузка STL-файла
просмотр статуса заказа
история заказов

Администратор:

просмотр всех заказов
изменение статуса заказов
управление пользователями
контроль выполнения заказов

Функционал проекта:

Авторизация:

регистрация пользователей
вход и выход из системы
разграничение ролей user/admin

Система заказов:

создание нового заказа
загрузка STL-файла
автоматический расчет стоимости
привязка заказа к пользователю

CRM-система (админ-панель):

таблица всех заказов
изменение статуса заказа: new, in_progress, printing, done, shipped
фильтрация и поиск заказов

Работа с файлами:

загрузка STL-моделей
хранение файлов на сервере
привязка файлов к заказу

REST API:

GET /api/orders — список всех заказов
GET /api/orders/<id> — информация о заказе
POST /api/orders — создание заказа
POST /api/calc_price — расчет стоимости заказа

Используемые технологии:
Python 3, Flask, Flask-SQLAlchemy (ORM), Flask-Login, SQLite, HTML, CSS, Bootstrap, REST API, работа с файлами
