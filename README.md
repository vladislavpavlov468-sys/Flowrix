# Flowrix Market CRM (2026)

**Описание проекта:**
Flowrix Market CRM — это веб-приложение на Flask, представляющее собой маркетплейс товаров, изготовленных на 3D-принтере, с интегрированной CRM-системой для управления заказами, товарами и пользователями.
Пользователи могут просматривать каталог товаров, оформлять заказы и отслеживать их статус, а администратор управляет товарами, заказами и клиентами через CRM-панель.

## Цель проекта:
Разработать веб-приложение, которое:
- реализует онлайн-магазин товаров
- позволяет оформлять и отслеживать заказы
- предоставляет администратору инструменты управления (CRM)
- использует базу данных, API и систему авторизации

## Роли пользователей

**Пользователь (client):**
- регистрация и авторизация
- просмотр каталога товаров
- просмотр карточек товаров
- добавление товаров в корзину (заказ)
- оформление заказа
- просмотр истории заказов
- отслеживание статуса заказа

**Администратор (admin):**
- управление товарами (добавление, редактирование, удаление)
- просмотр всех заказов
- изменение статусов заказов
- управление пользователями
- просмотр статистики

## Функционал проекта

**Авторизация:**
- регистрация пользователей
- вход и выход из системы
- разграничение ролей (user/admin)

**Каталог товаров:**
- отображение списка товаров
- карточка товара (название, описание, цена, изображение)
- фильтрация и поиск товаров

**Система заказов и корзина:**
- добавление товаров в корзину с подсчётом суммы (AJAX)
- оформление заказа
- автоматический расчет стоимости
- привязка заказа к пользователю

**CRM-система (админ-панель):**
- таблица всех заказов
- изменение статуса заказа:
  - `new` - новый (корзина)
  - `in_progress` - в обработке
  - `shipped` - отправлен
  - `delivered` - доставлен
- управление товарами
- фильтрация заказов
- поиск пользователей и заказов

**Работа с файлами:**
- загрузка изображений товаров (обработка через Pillow)
- хранение файлов на сервере
- привязка изображений к товарам

**REST API:**
- `GET /api/cart` — получение данных корзины
- `DELETE /api/cart/<item_id>` — удаление товара из корзины
- `GET /api/products` — список товаров
- `GET /api/products/<id>` — информация о товаре
- `GET /api/orders` — список заказов
- `GET /api/orders/<id>` — информация о заказе
- `POST /api/orders` — создание заказа

## Используемые технологии:
- Python 3
- Flask
- Flask-SQLAlchemy (ORM)
- Flask-Login
- SQLite
- HTML / CSS / Bootstrap 5 / JavaScript (AJAX)
- REST API
- Pillow (работа с изображениями)

## Структура базы данных

- **Users:** `id`, `username`, `email`, `password_hash`, `role`, `created_at`
- **Products:** `id`, `name`, `description`, `price`, `image_filename`, `category`, `is_available`, `created_at`
- **Orders:** `id`, `user_id`, `total_price`, `status`, `created_at`
- **OrderItems:** `id`, `order_id`, `product_id`, `quantity`, `price_at_purchase`
