Flowrix3D Market CRM - маркетплейс товаров для 3D-печати с CRM-системой
Описание проекта:
Flowrix3D Market CRM - это веб-приложение на Flask, представляющее собой маркетплейс товаров для 3D-печати с интегрированной CRM-системой для управления заказами, товарами и пользователями.
Пользователи могут просматривать каталог товаров, оформлять заказы и отслеживать их статус, а администратор управляет товарами, заказами и клиентами через CRM-панель.

Цель проекта:
Разработать веб-приложение, которое:
реализует онлайн-магазин товаров
позволяет оформлять и отслеживать заказы
предоставляет администратору инструменты управления (CRM)
использует базу данных, API и систему авторизации
Роли пользователей

Пользователь (client):
регистрация и авторизация
просмотр каталога товаров
просмотр карточек товаров
добавление товаров в заказ
оформление заказа
просмотр истории заказов
отслеживание статуса заказа

Администратор (admin):
управление товарами (добавление, редактирование, удаление)
просмотр всех заказов
изменение статусов заказов
управление пользователями
просмотр статистики
Функционал проекта

Авторизация:
регистрация пользователей
вход и выход из системы
разграничение ролей (user/admin)

Каталог товаров:
отображение списка товаров
карточка товара (название, описание, цена, изображение)
фильтрация и поиск товаров

Система заказов:
добавление товаров в заказ
оформление заказа
автоматический расчет стоимости
привязка заказа к пользователю

CRM-система (админ-панель):
таблица всех заказов
изменение статуса заказа:
new - новый
in_progress - в обработке
shipped - отправлен
delivered - доставлен
управление товарами
фильтрация заказов
поиск пользователей и заказов

Работа с файлами:
загрузка изображений товаров
хранение файлов на сервере
привязка изображений к товарам

REST API:
GET /api/products — список товаров
GET /api/products/<id> — информация о товаре
GET /api/orders — список заказов
GET /api/orders/<id> — информация о заказе
POST /api/orders — создание заказа
